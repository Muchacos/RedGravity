# RedGravity

![](https://s3.image.hosting/2021/03/18/j6hs.jpg)
Это игра в которой вам нужно управлять марсоходом и выполнять задания на время!

## О проекте
RedGravity создан за месяц в качестве проекта для RTUITLab (2 уровень). Игровой движок - UnrealEngine 4. Основа класса для марсохода написана на C++, а остальной код выполнен на Blueprint. Структура проекта организовывалась в соответсвии с [CleanCut UE4StyleGuide](https://github.com/CleanCut/UE4StyleGuide). Управление - клавиатура или геймпад (реализована нативная поддержка геймпада Dualshok4).

## Описание игрового процесса
В этой игре вам предстоит за определенное время выполнить несколько поставленных задач. Это может быть исследование температуры на поверхности Марса, взятие образцов грунта и т.д. За каждое задание вы будете получать очки. Чтобы выполнить цель, нужно заехать в ее зону выполнения (расстояние до цели станет красным) и зажать "F". Посмотреть локации целей можно на большой карте (открывается по клавише "M"). Но недостаточно просто провести исследования! Как только вы сделаете их нужное количество, вам предстоит отправить полученные данные исследований. Т.к. сигнал с Марса достаточно плохой, данные отправляются с большой горы в центре карты:
 
 ![Гора](https://s3.image.hosting/2021/03/17/jCR7.jpg)

Не пропустите момент, когда появится задание "Отправить данные"! Это финальное задание, которое завершает игру. Победа ваша, если вы успеете выполнить нужное количество заданий и отправить из данные с горы.

Если вы перевернулись, у вас есть возможность перевернуть марсоход обратно по нажатию клавиши "Q". Но это будет стоит вам очков!

## Управление
- W, A, S, D — управление ровером
- Мышь — Управление камерой
- Esc, P — пауза
- Space — Тормоз
- F — Выполнить цель
- Q — Перевернуть марсоход
- T — Выключить отображение целей
- M — Карта
- \+ или - — Масштаб миникарты
- 1 — Основной вид
- 2 — Вид с камеры ровера
- C — Переключение видов
