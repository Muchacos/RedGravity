# RedGravity

![](https://s3.image.hosting/2021/03/17/jTHV.png)
Это игра в которой вам нужно управлять марсоходом и выполнять задания на время!

## О проекте
RedGravity создан за месяц в качестве проекта для RTUITLab. Игровой движок - UnrealEngine 4. Основа класса для марсохода написана на C++, остальной код выполнен на Blueprint. Структура проекта организовывалась в соответсвии с [CleanCut UE4StyleGuide](https://github.com/CleanCut/UE4StyleGuide).

## Установка и игра
Скачать [билд](https://drive.google.com/drive/folders/1JWNNKm6T6Q55F2nsZzsaaFpgCl0AGr3F?usp=sharing) и запустить RedGravity.exe

## Описание игрового процесса
 В этой игре вам предстоит за определенное время выполнить несколько поставленных задач. Это может быть исследование температуры на поверхности Марса, взятие образцов грунта и т.д. За каждое задание вы будете получать очки. Чтобы выполнить цель, нужно заехать в ее зону выполнения (расстояние до цели станет красным) и зажать "F". Но недостаточно просто провести исследования! Как только вы сделаете их нужное количество, вам предстоит отправить полученные данные исследований. Т.к. сигнал с Марса достаточно плохой, данные отправляются с большой горы в центре карты:
 
 ![Гора](https://s3.image.hosting/2021/03/17/jCR7.jpg)

Не пропустите момент, когда появится задание "Отправить данные"!
Посмотреть локации целей можно на большой карте (открывается по клавише "M")
Если вы перевернулись, у вас есть возможность перевернуть марсоход обратно по нажатию клавиши "Q". Но это будет стоит вам очков!

## Управление
- W, A, S, D - управление ровером
- Мышь - Управление камерой
- Esc, P - пауза
- Space - Тормоз
- F - Выполнить цель
- Q - Перевернуть марсоход
- T - Выключить отображение целей
- M - Карта
- \+ - Увеличить миникарту
- \- - Уменьшить миникарту
- 1, 2 - Переключение видов
